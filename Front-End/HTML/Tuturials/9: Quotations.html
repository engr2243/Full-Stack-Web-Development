Block Quotations(blockquote)

The HTML element blockquote is used element defines a section that is quoted from another source.
such as a webpage i.e

Here is a quote from WWF's website:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

Short quotation(q)

The HTML element q is used element define short quotation.
i.e

WWF's goal is to: Build a future where people live in harmony with nature.

Abbrevation(abbr)

The HTML element abbr is used to define abbrevations in html webpages.
i.e

The WHO was founded in 1948.

Address(address)

The HTML element address is used to define contact information for the author/owner of a document or an article.
The contact information can be an email address, URL, physical address, phone number, social media handle, etc.
The text in the address element usually renders in italic, and browsers will always add a line break
before and after the address element. i.e

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Citation(cite)

The HTML element cite The HTML tag defines the title of a creative work
(e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.) i.e

The Scream by Edvard Munch. Painted in 1893.
