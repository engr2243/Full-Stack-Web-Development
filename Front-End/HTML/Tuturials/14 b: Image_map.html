Image Map

  • Image map allows to create on or more clickable areas on an an image for further exploring.
  • HTML tag(map) is used to create clickable areas on image
  • The idea behind an image map is that you should be able to perform different actions depending on where in the image you click.
    •

    Working of image maps

  • The image is inserted using the 'img' tag. The only difference from other images is that you must add a usemap attribute starting from hash(#usemap)
  • Then, add a "map" element.The "map" element is used to create an image map, and is linked to the image by using the required name attribute having the same value as img's usemap attribute
  • Use area attribure to add the clickable areas. The area attribute can be passed as 'rect', 'circle', 'poly' and 'default'. 'default' means all image are area
  • Another attribute 'coord' is passed in pairs to specify the area
